# Controle de Vendas (v5.0)

## Rodar no computador
1) Instalar dependências:
```bash
npm install
```

2) Rodar local:
```bash
npm run dev
```

Abra o link que aparecer (ex: http://localhost:5173).

## Build para colocar online (Netlify)
```bash
npm run build
```
A pasta gerada será `dist/`.

Este projeto já inclui `netlify.toml` com o comando/pasta corretos.
